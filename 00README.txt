# This file explains the contents of this directory which contains
# supplementary data for the paper "Ribovore: ribosomal RNA sequence
# analysis for GenBank submissions and database curation".
#
# Files that end in .gz can be unpacked with the command 'gunzip
# filename'
# 
# =====================================================================
#
# Files in this directory:
# 
# ribodbmaker-tests.sh: shell script with explanatory comments and
# instructions for running the ribodbmaker tests from the Ribovore paper.
#
# rnammer-tests.sh: shell script with explanatory comments and
# instructions for running the RNAmmer tests from the Ribovore paper.
#
# yarza.fa.gz, yarza.tax.txt.gz: input files for the yarza tests, see
# the .sh files for details.
#
# silvaparc.fa.gz: input file for the silvaparc test, see the .sh
# files for details.
#
# microsporidia.fa.gz: input file for the microsporidia test, see the
# .sh files for details.
#
# silvaparc-and-microsporidia.tax.txt.gz: input file for the
# silvaparc and microsporidia tests, see the .sh files for details.
#
# rnammer.yarza.hmmreport.gz, rnammer.yarza.pass.fa.gz: expected
# output files for the RNAmmer yarza tests, see rnammer-tests.sh for
# details.
#
# rnammer.silvaref.hmmreport.gz, rnammer.silvaref.pass.fa.gz: expected
# output files for the RNAmmer silvaref tests, see rnammer-tests.sh for
# details.
#
# 00README.txt: this file.
# 
# ====================================================================
